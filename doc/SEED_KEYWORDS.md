## Покриття кластерів (поточний стан)

```
✅ Покрито:
  - Комерційний: 3 landing pages (nextjs, landing, ai)
  - Інформаційний: 3 гайди (ціна, вибір розробника, React vs Tilda)
  - Доказовий: 4 кейси + 2 devlog
  - AI-ніша: landing + стаття Jarvis

❌ Не покрито (пріоритет):
  - Локальний: жодної сторінки "Київ" → Sprint E
  - Технічний: жодного туторіалу (Next.js SEO, CWV) → Sprint C-D
  - Нішеві landing: юристи, ресторани, освіта → Sprint C+
  - Hub page: немає єдиної "послуги" hub-сторінки
```

Про клієнта

ГО «Академія сучасних освітян» — це офіційний суб'єкт підвищення кваліфікації педагогічних працівників з Білої Церкви (Київська обл.). За 12+ років роботи Академія підготувала більше 8 700 педагогів і видала стільки ж офіційних сертифікатів, які зараховуються при атестації за Постановою КМУ №800.

Засновниця — Жанна Вихристенко, кандидат педагогічних наук, сертифікований тренер-коуч НУШ, арт-коуч, ігротехнік, доцент кафедри педагогіки і психології КОІПОПК. З 2012 року допомагає тисячам педагогів із підвищенням кваліфікації та впровадженням сучасних практик.

Задача

Старий сайт Академії перестав відповідати масштабу організації. Потрібна була не просто «нова обгортка», а повноцінна платформа, яка:

  • Одразу показує офіційний статус та довіру — щоб вчитель не сумнівався
  • Дозволяє зручно обирати програми підвищення кваліфікації (курси 30–150 год, вебінари 6–16 год, тренінги 16 год)
  • Має реєстр сертифікатів — перевірку виданих документів онлайн
  • Містить блог із методичними матеріалами
  • Працює швидко, виглядає сучасно, відмінно індексується пошуковиками
Особливість проєкту: цільова аудиторія — вчителі, які часто скептично ставляться до онлайн-курсів. Сайт мусить із першого погляду переконати: це офіційно, це якісно, сертифікат дійсно зараховується.

Що було vs Що стало

✗ Було

Застарілий дизайн, повільне завантаження, відсутня мобільна адаптація, немає реєстру сертифікатів, контент складно оновлювати, SEO не оптимізований

✓ Стало

Сучасна платформа на Next.js, миттєве завантаження, повна адаптивність, реєстр сертифікатів, зручне управління контентом, повна SEO-оптимізація

Хід розробки

01

Аналіз аудиторії

Вивчив потреби цільової аудиторії — вчителів ЗЗСО. Головне: довіра до офіційності, простота вибору програми, мобільний доступ.

02

Дизайн та UX

Створив дизайн-систему з «академічними» кольорами: стриманий teal + золотий акцент. Фокус на довірі: сертифікати, цифри, відгуки — одразу на першому екрані.

03

Архітектура та розробка

Next.js 14 з App Router, TypeScript, Tailwind CSS. Окремі модулі: каталог програм, реєстр сертифікатів, блог, контактна форма.

04

SEO та оптимізація

Schema.org розмітка, семантична структура H1–H6, мета-теги для кожної сторінки, sitemap.xml, robots.txt. Оптимізація зображень у WebP.

05

Деплой на Railway

CI/CD через GitHub → Railway. Автоматичний білд при кожному пуші. Node.js 22, npm ci, next build & start.

Технічний стек

Frontend

Next.js 14, React, TypeScript, Tailwind CSS. SSR для SEO, оптимізація зображень через next/image.

Стилі та анімації

Tailwind CSS з кастомною дизайн-системою. Framer Motion для плавних анімацій та мікровзаємодій.

Хостинг

Railway — автоматичний деплой з GitHub, Node.js 22, CI/CD, автоскейлинг. Домен www.ace.org.ua через DNS.

SEO & Analytics

Повна Schema.org розмітка (Organization, Course, BreadcrumbList), Open Graph, структуровані дані для пошуку.

Результат

Нова версія сайту www.ace.org.ua запущена в березні 2026. Сайт вже працює на домені Academy, прийнятий замовником, і обслуговує тисячі вчителів по всій Україні. За відгуками засновниці — вчителі нарешті можуть «за 30 секунд обрати програму і зареєструватись».

Це проєкт, де важливо не тільки «зробити гарно», але й зробити переконливо — щоб вчителька з Фастова, яка бачить сайт вперше, одразу зрозуміла: це офіційно, це зараховується, це варто.
